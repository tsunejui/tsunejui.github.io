How to export kubeconfig file from existing cluster? 如何匯出kubeconfig file 從你的cluster

工作環境上我們會需要從其他電腦存取k8s API, 已便使用kubectl等工具操作我們的cluster, 可以透過下面的方法達成:

準備工作

env:

  • A Server
  1. 到 A Server 上使用 cat ~/.kube/config 查看 kubeconfig.
  2. 使用kubectl proxy 對外開放k8s API, 打開後可以 curl 127.0.0.1:8080 確認是否正常運作。
  • Client
  1. 到 Client 上修改 ~/.kube/config 檔案
  2. 貼上來自 A Server 的 clusters, contexts, users 內容。

確認

使用 kubectl get nodes 查看是否已經正確連上。
