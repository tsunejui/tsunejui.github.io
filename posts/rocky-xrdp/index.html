Rocky Linux 使用 XRDP

Rocky Linux XRDP 安裝步驟

April 30, 2023
Linux

這篇文章將介紹如何在 Rocky Linux 上安裝和配置 XRDP，一種遠程桌面協議。

使用環境

Rocky Linux 8.7

安裝 xrdp server

安裝 xrdp:

dnf install xrdp

啟動 server

啟動 xrdp server:

systemctl start xrdp

設定開機啟動:

systemctl enable xrdp

打開防火牆

預設 port 為 3389:

firewall-cmd --permanent --add-port=3389/tcp && firewall-cmd --reload
