Rocky 使用 XRDP

使用環境

Rocky Linux 8.7

安裝 xrdp server

安裝 xrdp:

dnf install xrdp

啟動 server

啟動 xrdp server:

systemctl start xrdp

設定開機啟動:

enable xrdp

打開防火牆

預設 port 為 3389:

firewall-cmd --permanent --add-port=3389/tcp && firewall-cmd --reload
