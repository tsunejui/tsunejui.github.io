產生新的加入 cluster 指令

建立新的 token

kubeadm token generate

畫面會顯示新的 token, 便可以使用 kubeadm token create 建立新的 join command

以 4a4dv5.i31huo1je1e2dler 為例

kubeadm token create 4a4dv5.i31huo1je1e2dler --print
